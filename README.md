# StickerSprint 2026

A standalone web app for tracking your Panini FIFA World Cup 2026 sticker album collection.

## Features

- 📱 Works on phone and desktop
- 🔄 Auto-synced between devices (localStorage)
- 🎯 Track owned stickers, duplicates, and special parallel borders
- 💰 Monitor spending on booster packs
- 🏆 View match fixtures and scores
- 📊 Analyze your collection stats

## Usage

Simply open `StickerSprint-2026.html` in your browser on any device. Your data syncs automatically via browser localStorage.

### Export & Import

- **Export**: Dashboard → Export button to backup your data
- **Import**: Dashboard → Import button to restore from backup

## Browser Support

Works on all modern browsers (Chrome, Firefox, Safari, Edge) on desktop and mobile.

## Data

Your sticker data is stored locally in your browser and never sent to any server.
